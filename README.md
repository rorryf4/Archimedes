# Archimedes

## Development

### Running the web app

From the repo root:

- Start dev server: `pnpm dev:web`
- Build: `pnpm build:web`
- Lint: `pnpm lint:web`
- Typecheck: `pnpm typecheck:web`

### Verification

To run the standard verification pipeline (lint + typecheck):

```bash
pnpm verify
```
